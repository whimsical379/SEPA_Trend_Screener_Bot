# SEPA Trend Screener Bot (v1.0)

## 1. 项目简介
本项目是一个基于 Python 开发的自动化美股筛选系统，核心逻辑严格遵循传奇交易大师 **Mark Minervini** 的 **SEPA (Specific Entry Point Analysis)** 趋势模板。

系统通过 Finviz 进行多因子初筛，随后利用 `yfinance` 获取高频数据进行三周期（周线、日线、小时线）深度校验，最终将符合“趋势模板”且具备入场潜力的标的通过微信推送至用户。

## 2. 策略逻辑 (Strategy Logic)
系统通过三个维度的过滤确保标的处于强势增长阶段：

### 2.1 趋势模板过滤 (Trend Template)
* **均线排列**：价格 > MA50 > MA150 > MA200。
* **相对强度 (RS)**：个股表现必须在 20 日和 250 日周期内均跑赢标普 500 指数 (SPY)。
* **形态验证**：要求 MA200 持续向上至少 1 个月，且价格处于 52 周高点的 25% 范围内。

### 2.2 三周期校验系统
* **周线级别 (Gate)**：验证长期均线的一致性，防止在大周期乖离率过大时盲目入场。
* **日线级别 (Core)**：核心筛选层，包含成交量放量校验、RSI 超买风控以及前 10 日高点突破确认。
* **小时线级别 (Timing)**：辅助择时，识别短期回踩均线后的低吸机会或警示严重超买。

## 3. 技术核心参数
您可以直接在 `sepa_screener_bot.py` 的“全参数配置区”进行自定义修改：

| 参数名称 | 默认值 | 说明 |
| :--- | :--- | :--- |
| `MARKET_CAP_MIN` | 5,000M | 最小市值（50亿美元） |
| `EXCLUDE_INDUSTRIES` | Banks, Utilities... | 排除高股息/低波动行业 |
| `WEEKLY_BIAS_LIMIT` | 1.4 | 周线 MA50 最大允许乖离率 |
| `DAILY_RSI_LIMIT` | 70 | 日线 RSI 风险阈值 |
| `STOP_LOSS_PCT` | 0.07 | 初始硬止损比例 (7%) |

## 4. 自动化部署 (Workflow)
本项目实现了完全的无服务器化运行：
* **运行环境**：GitHub Actions (`ubuntu-latest`)。
* **执行时间**：每日北京时间 06:00（UTC 22:00），确保美股收盘数据已完整同步。
* **时区优化**：代码内置 `Asia/Shanghai` 时区转换，推送的时间戳符合国内查看习惯。

## 5. 快速开始
1. **Fork 本仓库**。
2. **配置 Secrets**：在 GitHub 仓库设置中，添加 `SERVER_CHAN_SENDKEY`，填入您的 Server 酱 Key。
3. **手动触发测试**：在 Actions 页面点击 `workflow_dispatch` 即可立即执行并检查微信推送。
